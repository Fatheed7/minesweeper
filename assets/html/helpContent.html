Help clear all the mines!
  • Click on a cell to reveal it.
  • If it's empty, you'll see how many of the neighbouring cells contain bombs.
  • But beware! If it's a bomb, all the bombs will explode!
  • Right click to place a flag on a cell you suspect to be a bomb.
  • Right click again to remove it.
  • When only cells containing bombs remain, you win!
  • If you make the mines explode, you lose!
  • The top bar of the game page shows:
  • Bombs - How many bombs the current difficulty contains.
  • Flag - How many flags you have left (You start with a flag for each bomb).
  • Time - How long you've been playing (in seconds).
  • The bar at the bottom of the page shows difficulty settings,which are:
  • Beginner - A 9 x 9 grid containing 10 bombs.
  • Intermediate - A 16 x 16 grid containing 40 bombs.
  • Expert - A 30 x 16 grid containing 99 bombs.
  • Click the icon in thebottom left corner to disable any animation effects.
  • Click the icon in the bottomright corner to view these instructions again.
